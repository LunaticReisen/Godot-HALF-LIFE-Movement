# GodotSourceMovment

使用 Godot 4.2.1

使用 __Godot Jolt__ 和 __Func_Godot__  插件

## 已实现的功能

- [x] 兔子跳
- [x] 滑翔
- [x] Air Strafe [(空中走位?](https://wiki.teamfortress.com/wiki/Strafing/zh-hans)反正就是旋转跳啥的身法)
- [x] 蹲伏
- [x] 滚轮跳
- [x] 蹲跳
- [x] 爬楼梯
  - [x] 平滑镜头
  - [x] 蹲伏状态下爬楼梯
- [x] 梯子
- [x] 游

### 附加内容

- [x] 物体交互
